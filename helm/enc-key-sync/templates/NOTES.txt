image that we have created using the key.

Install a key by creating a generic secret in the `enc-key-sync` namespace,
with `--type=key`:

```
$ kubectl create -n enc-key-sync secret generic \
    --type=key \
    --from-file=my-priv-key.pem \
    my-decryption-key
```
